## 🚀 Deployment Instructions

### For End Users

1. **Visit the project:**
   - Go to project repository
   - Read 00-START-HERE.md
   - Follow GETTING_STARTED.md

2. **Install:**
   ```bash
   curl -L [...]/umu-game-installer.sh -o ~/.local/bin/umu-game-installer
   chmod +x ~/.local/bin/umu-game-installer
   ```

3. **Use:**
   ```bash
   umu-game-installer game-setup.exe
   ```

### For Distributors

1. **Package the scripts:**
   - umu-game-installer.sh → /usr/bin/
   - umu-game-manager.sh → /usr/bin/
   - Documentation → /usr/share/doc/umu-game-installer/

2. **Create package:**
   - Debian: .deb package
   - Arch: PKGBUILD
   - Flatpak: Flatpak manifest

3. **Dependencies:**
   - umu-run (required)
   - bash (required)
   - coreutils (required)
